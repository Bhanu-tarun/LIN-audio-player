I was inspired by this video I once saw, that inspired me to decentralise from apple music and spotify. I wanted to intentionally listen to my music not just have it be background noise for whatever I'm doing. 

I really like zune's interface, but I didn't want to buy a zune or use it for my daily music listening. I figured I could make my own software to have the same feel, but with my own twist and control over my music. 

im using react for the frontend and fastapi for the backend. considering their usability and ease of learning, theyre great for a beginner like me. and also just plain css because i really dont need frameworks. 

currently the project uses a uvicorn server, but i plan on replacing it with a more production ready server like gunicorn. 

as for now, this project is for personal use only, but i plan on making it more widely available in the future. 

it has a few features, like:
- tagging music
- album art editing
- album organization
- playlisting
- queueing songs    
- (planning on adding a background that reacts to how the music sounds, gonna take me a little to figure it out)
- automatic metadata handling.
